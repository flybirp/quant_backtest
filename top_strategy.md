## PB/缩量 衰减边界

```
PB轴(vc=0.9):
PB<2 → PB<3:  +41笔 (-5.6pp EV)  ← 性价比最高点
PB<3 → PB<4:  +15笔 (-2.8pp EV)  ← 衰减开始
PB<4 → PB<5:  +12笔 (-1.3pp EV)  ← 不划算
PB<5 → ∞:    +14笔 (-1.0pp EV)  ← 无意义

缩量轴(PB<2):
vc=0.7 → vc=0.85: +33笔 (-5pp EV)
vc=0.85→ vc=0.9:  +3笔  (+0.4pp EV)  ← 最优
vc=0.9 → 无vc:    +75笔 (-9pp EV)   ← 崩盘
```

### 关键发现

1. **大盘恐慌过滤器**：hs300 快速杀跌(10天>5%) > 深度回撤(252天>10%)
2. **缩量是最强质量信号**：vc 从无到 0.9，EV 从 27%→36%（+9pp），笔数仅减 75
3. **vol>2 不可放松**：降到 1.5 反而 EV 下降
4. **PB<3 是拐点**：再放宽边际收益为负
5. **中证1000 替代无效**：单独使用不如 hs300 快速杀跌+缩量组合

### zzh关键发现

1. 支撑衰竭是毒药 — 移除后 EV 从 +1% 跳到 +1.8%
2. 精选 > 数量 — 收紧入场 EV 翻 4.5 倍

4. 标的质量敏感 — 大蓝筹 EV+6.18%，全量掉到 +2.15%
5. 76%水下是结构性代价 — 不可修复
