## 📊 关键发现

### 1. 恐慌错杀系列统治 A 级
r13~r16 和 RS10 系列占据前6，核心优势是 **低回撤(11-13%) + 高EV(15-23%) + 低月交易(1-4笔)**，散户可执行性极强。

### 2. A-口袋支点-激进-分批 是"异类"
CAGR 高达 82%、Calmar 8.56（全场最高），但月交易 308 笔，可执行性扣分严重。适合程序化交易，不适合散户。

### 3. DS 系列是"隐形冠军"
DS2d/DS6/DS7/DS8/DS9 全部在 A 级，EV 高（15-32%）、持仓长（500-700天），但交易极稀疏（月均 1-3 笔），适合耐心型投资者。

### 4. 分批策略普遍加分
分批建仓版本（A-激进-分批、V1-激进-分批、F1-金叉回踩分批）比原版得分更高，主要因为夏普和 Calmar 提升。

### 5. 月交易 > 50 笔的策略都被"可执行性"拖累
H2/G2/F1-v2 等策略 CAGR 很高（60%+），但月交易 60-95 笔，得分只有 50-57。

## 评分公式

```python
score = 0
# Calmar: 30分 (CAGR / MaxDD, >1 满分)
score += min(calmar / 15 * 30, 30)
# EV/MaxDD: 25分 (每承受1%回撤的期望收益, >0.5 满分)
score += min(max(ev / max_dd, 0) / 1 * 25, 25)
# 胜率×盈亏比: 10分
score += min(win_rate * pl_ratio / 1 * 10, 10)
# 可执行性: 10分 (月均交易越少越好)
if monthly_trades <= 20: score += 10
elif monthly_trades <= 50: score += 7
else: score += 3
# 夏普: 25分 (>1.5 满分)
score += min(max(sharpe, 0) / 1.5 * 25, 25)
```
