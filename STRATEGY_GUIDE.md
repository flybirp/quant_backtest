# 上班族策略寻优指南

## 目标画像

挂完买入单 + 止盈止损单就去上班，不需要盯盘。要求：

- **低频**：月均 1-5 笔信号，午休看一眼够用
- **明确退出**：固定止盈止损，不依赖移动止损或复杂条件
- **扛得住**：最大回撤 < 20%，不会开会回来发现腰斩
- **拿得住**：胜率 > 60%，连续亏损 < 5 笔，心理上扛得住
- **持仓周期**：20-80 天，不会今天买明天卖

## 三、策略寻优流程

如果觉得 zzhY.3 信号太少（月均 1.3 笔），可以自己调参。核心可调参数在 `strategies/rule/zzhY.3.yaml` 的 `state_machine_params` 里：

```yaml
state_machine_params:
  entry_stabilize_days: 7    # 买入前需站稳天数 — 调小=更多信号
  high_pos_pct: 95           # 高位阈值 — 调小=更早止盈
  low_pos_pct: 99            # 低位阈值 — 调小=更多买入机会
  support_stuck: 99          # 支撑衰竭 — 调大=更少卖出
  trend_broken_days: 30      # 趋势破坏天数 — 调大=更少止损
  vol_rank_pct: 99           # 带量阈值 — 调小=更多信号
  add_pullback_pct: 3.0      # 加仓回调幅度
  smart_stop: false          # 智能止损开关
```

### 3.1 调参三步法

**Step 1：粗扫 — 找到 EV > 0 的参数范围**

```bash
make backtest S=zzhY.3 P=大蓝筹

# 改一个参数，比如 low_pos_pct 从 99 → 90 → 80 → 70
python report.py results/zzhY.3_v2_大蓝筹.json results/zzhY.3_v3_大蓝筹.json \
    --quick

# 看哪个版本 EV 和交易笔数最平衡
```

**Step 2：细调 — 在工作区间内用 sensitivity 扫描**

```bash
python report.py results/zzhY.3_大蓝筹.json \
    --sensitivity low_pos_pct \
    --values "70,75,80,85,90,95,99" \
    --strategy zzhY.3 --pool 大蓝筹
```

**Step 3：验证 — Walk-Forward 确认样本外有效**

```bash
python report.py results/zzhY.3_大蓝筹.json \
    --walk-forward --strategy zzhY.3 --pool 大蓝筹
```

### 3.2 上班族优化目标

不是追求最高 EV，而是找这四个指标的平衡点：

| 指标 | 目标 | 为什么 |
|------|------|--------|
| 月均交易 | 2-5 笔 | 太少怕踏空，太多看不过来 |
| 最大回撤 | < 20% | 10 万本金最多亏 2 万 |
| 胜率 | > 65% | 连续亏 3 笔还能有信心 |
| EV | > 3% | 年化能跑赢理财 |

找到满足这四点且 Walk-Forward WFE > 60% 的参数组合，就是你的配置。

## 四、实盘操作流程

### 4.1 每日操作（5 分钟，午休做）

```
1. 跑一遍选股 → 看今天有没有买入信号
2. 有信号 → 挂条件单（次日开盘价买入）
3. 同时挂止盈止损单：
   - 止盈: 买入价 × 1.15  （+15%）
   - 止损: 买入价 × 0.85  （-15%）
4. 看看现有持仓有没有触发退出 → 有就手动平
```

### 4.2 盘后（周末做，15 分钟）

```bash
# 更新持仓的止盈止损价（如果股票涨了）
# 看一眼最近一周的信号总结
make report S=zzhY.3
```

### 4.3 月度检查（30 分钟）

```bash
# 跑完整的基准对比 + 情景压力，确保策略没失效
make report-full S=zzhY.3
```

### 4.4 季度大检（1 小时）

```bash
# 全策略对比，看 zzhY.3 是否还是最优
make report-all

# 如果连续 2 个季度跑输基准 > 10%，考虑换策略或调参
```

## 五、资金管理建议

### 5.1 仓位分配

```
总资金 100%
├── zzhY.3: 40%  （主力，高胜率狙击）
```

zzh7.3 虽然 EV 不错但不适合上班族——月均 30 笔交易需要频繁操作。如果需要更多交易机会，用参数调松的 zzhY.3 v2 版本替代。

### 5.2 单笔仓位

- 单笔不超过总资金的 10%
- 最大同时持仓：3-5 只
- 止损触发后不补仓

## 六、行为纪律清单

| 规则 | 原因 |
|------|------|
| 信号来了必须执行 | 错过 1 笔可能 = 错过 +18% |
| 止损到了必须砍 | 一次不止损可能回撤 30%+ |
| 不在盘中临时改策略 | 盘中情绪不稳定 |
| 连续亏 5 笔停一周 | 策略可能暂时失效 |
| 季度跑输基准 10%+ 停用并复盘 | 可能需要调参 |
| 不追涨杀跌 | zzhY.3 是回调买入型，追涨反而亏 |
| 不止盈变止损 | +15% 到了就走，不要贪 |

## 七、备选方案

如果 zzhY.3 信号太少（年化 16 笔），备选：

| 策略 | 月均 | EV | 回撤 | 适合人群 |
|------|------|-----|------|----------|
| zzhY.4 | 2.0 笔 | +5.12% | -15.0% | 希望多一点交易 |
| zzhY.3-loose | ~5 笔 | ~4% | ~20% | 调松 low_pos_pct 后的版本 |
| zzh7.3 | 30 笔 | +4.15% | -15.6% | 每天有空操作的 |

调参公式：把 `low_pos_pct` 从 99 降到 80，`vol_rank_pct` 从 99 降到 90，信号量大约翻 3 倍，EV 大约降 1-2 个百分点。
